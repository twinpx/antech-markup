Гофротара — вид товарной упаковки из гофрированного листа. Может быть индивидуальной, массовой, транспортной. Производим упаковку из гофоркартона всех групп и типов, предлагаем готовые изделия, разработку дизайна индивидуальной тары, включая:

  • паллетные короба для погрузки на поддоны;
  • четырехклапанные коробки для ручной сборки и формовочных автоматов;
  • самосборные боксы из одного листа, телескопические с крышкой;
  • штучную мини-упаковку для сыпучих и фасованных товаров.

Компания использует стандартный и микрогофрокартон. Возможно изготовление из белой и бурой гофры картонной упаковки для пищевой продукции.
