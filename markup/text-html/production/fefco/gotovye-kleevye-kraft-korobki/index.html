Готовые крафт-коробки для упаковки по FEFCO 07хх изготавливаются из неотбеленного гофрокартона, собираются из одного листа при помощи клея. Могут иметь разные размеры и конструкцию. Антэк производит крафтовые коробки для хранения и презентации товаров. В наличии и под заказ тара для бутылок, стаканов, автозапчастей, подарочные коробки, гофроящики для игрушек, книг, обуви и другие виды продукции. Возможно изготовление по образцу, разработка конструкции и дизайна в срок от 1 дня.
