Один из базовых вариантов упаковки — коробка из плотного картона. В зависимости от типа продукции, особенностей складского хранения и транспортировки, заказчики выбирают тару из 2-х, 3-х, 5-ти и 7-слойного гофрокартона.

Для синхронизации логистических процессов и облегчения идентификации тары, Европейская ассоциация производителей гофротары приняла классификатор (каталог FEFCO), разделяющий стандартные картонные коробки на следующие группы:

  • FEFCO 02xx — типовые складные коробки из одного листа;
  • FEFCO 03xx — складные картонные коробки из двух и более частей (коробка + крышка);
  • FEFCO 04xx — изделия из одного листа, собираемые без клея, скоб, липкой ленты;
  • FEFCO 05xx — сборная тара, собираемая с помощью клея, скоб, липкой ленты;
  • FEFCO 06xx — жесткий короб с корпусом и отдельными боковыми стенками, собираемый с помощью клея, скоб, липкой ленты;
  • FEFCO 07xx — готовые проклееные короба из одного листа;
  • FEFCO 09xx — вспомогательные упаковочные материалы, в том числе перегородки, вкладыши, решетки, амортизаторы и др, которые могут использоваться и отдельно.
